អារម្ភកថា

វិស័យអប់រំដើរតួនាទីយ៉ាងសំខាន់ក្នុងការអភិវឌ្ឍធនធានមនុស្សប្រកបដោយគុណភាពទាំងជំនាញរឹង និងជំនាញទន់ ដើម្បីរួមចំណែកក្នុងការអភិវឌ្ឍប្រទេសជាតិឱ្យមានការរីកចម្រើន។ ក្នុងន័យនេះ សាលារៀនជាកន្លែងដែល ផ្តល់នូវជំនាញទាំងពីរខាងលើទៅដល់សិស្សានុសិស្សទាំងពីរភេទ ដោយមិនប្រកាន់ជីវភាព ជាតិសាសន៍ សាសនា ឫនិន្នាការនយោបាយផ្សេងៗ។ ដើម្បីឱ្យសាលារៀនមួយអាចដំណើរការទៅប្រកបដោយស័ក្តិសិទ្ធភាព និងប្រសិទ្ធភាព ទាមទារឱ្យសាលារៀននោះ មានអ្នកដឹកនាំមួយរឹងមាំក្នុងការត្រួសត្រាយជាតម្រាយ ដើម្បីបើកផ្លូវឆ្ពោះទៅសម្រេចគោលដៅសម្រាប់គ្រឹះស្ថានសិក្សាក្នុងវិស័យអប់រំប្រកបទៅដោយផ្លែផ្កា។ ដូច្នេះ សាលារៀនក្នុងឆ្នាំសិក្សា២០២៣-២០២៤ ត្រូវរៀបចំឡើងនូវយុទ្ធសាស្តនានាឲ្យស្របតាមគោលការណ៍ណែនាំ ពិសេស ការរៀបចំដោយមានការចូលរួមពីគ្រប់ភាគីពាក់ព័ន្ធ ក្នុងការធានាបាននូវគណនេយ្យភាព និងតម្លាភាព ព្រមទាំងការរៀបចំចំណុចដៅផ្អែកលើអាទិភាពប្រចាំឆ្នាំនីមួយៗរបស់សាលារៀន ឲ្យស្របតាមអាទិភាពរបស់ក្រុង ខេត្ត និងថ្នាក់ជាតិ។ ខ្ញុំសង្ឃឹមថា ឯកសារនេះ នឹងក្លាយជាឧបករណ៍ដ៏មានតម្លៃសម្រាប់សាលាកម្រិតបឋមសិក្សា ឱ្យកាន់តែប្រសើរ ពិសេសលទ្ធផលសិក្សារបស់សិស្សប្រកបដោយគុណភាព។ ទន្ទឹមនិងនេះ លទ្ធផលអាចសម្រេចបាន លុះត្រាតែមានការចូលរួមពីគ្រប់ភាគីពាក់ព័ន្ធទាំងអស់ក្នុងការគាំទ្រដល់ដំណើរការបង្រៀន និងរៀន។ ខ្ញុំសូមថ្លែងអំណរគុណចំពោះ លោកគ្រូ អ្នកគ្រូ គណៈកម្មការគ្រប់គ្រងសាលារៀន តំណាងអាណាព្យាបាលសិស្ស ព្រមទាំងភាគីពាក់ព័ន្ធនានាដែលបានចូលរួមសហការសកម្មភាពការងាររបស់សាលាសាលារៀន។

ប្រវត្តិគណៈគ្របគ្រងសាលាបឋមសិក្សាបឹងកុក

  • ពីឆ្នាំ១៩៧៩ - ១៩៨០ ( ២ឆ្នាំ ) ដឹកនាំដោយលោក តឹក វ៉េង (ស្លាប់)
  • ពីឆ្នាំ១៩៨១-១៩៨៤ (៤ឆ្នាំ)ដឹកនាំដោយលោកឯម ផាត (ស្លាប់)
  • ពីឆ្នាំ១៩៨១-១៩៨៤ (៤ឆ្នាំ)ដឹកនាំដោយលោក ស្វាយ សុភី (ស្លាប់)
  • ពីឆ្នាំ១៩៨១-១៩៨៤ (៤ឆ្នាំ)ដឹកនាំដោយលោក កុល គឹមឈឿន (មិនដឹងប្រវត្តិ)
  • ពីឆ្នាំ១៩៩៣-២០១២ (១៩ឆ្នាំ) ដឹកនាំដោយ លោក ព្រុំ សារិតមន្នី ( ស្លាប់​ )
  • ពីឆ្នាំ២០១២ - ២០១៨ (៧ឆ្នាំ) ដឹកនាំដោយលោក ផាន់ ប៊ុយរ៉ូ (ជាមន្រ្តីការិយាល័យបឋមនៃមន្ទីរ អប់រំ យុវជន និងកីឡា ខេត្តកំពង់ចាម)
  • ពីឆ្នាំ២០១៩ - ដល់បច្ចុប្បន្ន ដឹកនាំដោយលោកនាយក ហ៊ង ច័ន្ទធា ស្នាដៃក្នុងដំណែងដូចជាក្រាលការរ៉ូឡា កញ្ចុះ ស្ពានយោល របង ស្លាកសាលា ចាក់សាបជុំវិញអគារ ប្ដូរថ្នាំពណ៌ (អគារសិក្សា-បណ្ណាល័យ) ក្ដារខៀនកញ្ចក់ ផើងផ្កាលម្អ។

    • ទីតាំងសាលាបឋមសិក្សាបឹងកុក

    សាលាបឋមសិក្សាបឹងកុកមានទីតាំងស្ថិតនៅ ភូមិបឹងកុកទី២ សង្កាត់ បឹងកុក​ ក្រុងកំពង់ចាម ខេត្តកំពង់ចាម ប្រទេសកម្ពុជា។

    ចក្ខុវិស័យ និងបេសកកម្ម

    1. ចក្ខុវិស័យ

    សាលាបឋមសិក្សា បឹងកុក នឹងក្លាយជាសាលារៀនគំរូ ក្នុងការបណ្តុះបណ្ដាលធនធានមនុស្ស ឈានឆ្ពោះទៅរកការផលិតមូលធនមនុស្ស ប្រកបទៅ​ដោយ គុណភាព គុណតម្លៃ សមធម៌ តម្លាភាព យុត្តិធម៌ សុជីវធម៌ ការអភិវឌ្ឍ គណនេយ្យភាពសង្គម និងបរិយាបន្ន ដល់កុមារក្នុងគ្រឹះស្ថានសិក្សា ដើម្បីអាចរួមរស់នៅក្នុងសហគមន៍ប្រកបដោយសេចក្ដីថ្លៃថ្នូរ និងជាទំពាំងបន្តវេនរបស់ជាតិ។

    2. បេសកកម្ម

    ដើម្បីសម្រេចឲ្យបាននូវចក្ខុវិស័យខាងលើ សាលាមានបេសកកម្ម ដាក់ចេញនូវគោលការណ៍ដឹកនាំ និងគ្រប់គ្រងរាល់សកម្មភាពដូចជា៖

  • សុខភាពសិក្សា បរិស្ថាន អនាម័យ សន្តិសុខ សុវត្ថិភាព សណ្ដាប់ធ្នាប់ និងចំណីអាហារ
  • ការបង្រៀន និងរៀន (វិធីសាស្ត្របង្រៀន-អង្គទាំង៤របស់អ្នកសិក្សា សុ ចិ បុ លិ ៖ ស្ដាប់ គិត សួរ កត់ត្រា )
  • អនុវត្តគោលការណ៍មនុស្សសកល ស្អាត របៀប សុភាព ទៀងពេល សមាធិ ដើម្បីក្លាយជាមនុស្សអស្ចារ្យ
  • សហការល្អជាមួយគ្រប់ភាគីពាក់ព័ន្ធ ដើម្បីអភិវឌ្ឍសាលារៀនឱ្យក្លាយជាសាលារៀនគំរូ
  • អភិវឌ្ឍផ្នត់ឥរិយាបថ គ្រូ សិស្ស និងសហគមន៍ឲ្យមានការផ្លាស់ប្ដូរជាវិជ្ជមាន។
